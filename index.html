jQuery text and attribute

  • text() - sets or gets the text value of elements
  • html() - sets or gets the content (HTML format) of elements
  • val() - sets or gets the value of a form element
  • attr(attributeName) - to get the element's attribute value





jQuery text and update attribute value

  • text('text') - inserts new text into the element
  • html('htmlCode') - updates the element's HTML value
  • val('new_value') - updates the value of the form element
  • attr(attributeName, new_value) - updates the value of element attributes
