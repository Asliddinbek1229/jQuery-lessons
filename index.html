jQuery Lessons



jQuery CSS

  • addClass() - add one or more classes to elements;
  • removeClass() - remove one or more classes from elements;
  • toggleClass() - add or remove classes to elements;
  • css() - modifies style properties;


Other methods

  • width() - elementning kengligi;
  • height() - elementning balandligi;
  • innerWidth() - elementning kengligi, padding bilab birgalikda;
  • innerHeight() - elementning balandligi, padding bilan birgalikda;
  • outherWidth() - elementning kengligi, border bilan;
  • outherHeight() - elementning balandligi, border bilan birga;
  • outherWidth(true) - elementning kengligi, margin bilan;
  • outherHeight(true) - elementning kengligi, margin bilan;
